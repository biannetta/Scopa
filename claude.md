# LÖVE Card Game

A card game implementation built using the LÖVE (Love2D) framework and Lua.

## Project Structure

- `main.lua` - Entry point of the game
- `game.lua` - Main game logic and state management
- `player.lua` - Player class and related functionality
- `card.lua` - Card class implementation
- `deck.lua` - Deck management and card operations
- `classic.lua` - Object-oriented programming helper library
- `conf.lua` - LÖVE configuration file
- `assets/` - Directory containing game assets

## Dependencies

- LÖVE (Love2D) framework
- Classic library for OOP in Lua

## Getting Started

1. Install LÖVE (Love2D) from https://love2d.org/
2. Clone this repository
3. Run the game using:
   ```bash
   love .
   ```

## Project Overview

This is a card game implementation using the LÖVE framework. It utilizes object-oriented programming principles in Lua through the Classic library. The game manages players, cards, and decks while providing an interactive gaming experience.

### Key Components

- **Game**: Handles the main game loop and state management
- **Player**: Manages player states, hands, and actions
- **Card**: Represents individual cards with their properties
- **Deck**: Handles the collection and manipulation of cards
